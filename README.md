# RDP Config Updater

Este utilitário em C# automatiza a atualização de ficheiros `.rdp` na pasta de Downloads, garantindo que as definições de resolução e redimensionamento estão corretas.

## Funcionalidades
- Ativa o **Smart Sizing** (redimensionamento inteligente).
- Define a resolução para **1920x1080**.
- Configura o modo de ecrã para janela.
- **Inteligente**: Se a chave já existir, o valor é atualizado. Se não existir, é adicionada ao final do ficheiro.
